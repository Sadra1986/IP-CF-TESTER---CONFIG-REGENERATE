# 🌐 IP-CF-TESTER — CONFIG REGENERATE

ابزار تست و بازسازی کانفیگ‌های Cloudflare برای فیلترشکن‌های مبتنی بر V2Ray/Xray

-----

## 📌 این ابزار چیه؟

یه فایل HTML تک‌صفحه‌ای که **مستقیم روی مرورگر گوشی یا کامپیوتر** اجرا میشه و نیازی به نصب یا سرور نداره. کارش اینه که از بین هزاران IP کلودفلر، سریع‌ترین و بهترین‌ها رو با **نت خودت** پیدا کنه و باهاشون کانفیگ‌های جدید vless بسازه.

-----

## ⚙️ امکانات

- ✅ تست دسترسی به IP های Cloudflare با نت خودت
- ✅ بارگذاری خودکار لیست IP از IRCF (بیش از ۲۰۰۰ IP)
- ✅ پشتیبانی از آپلود فایل لیست IP یا کانفیگ
- ✅ فیلتر بر اساس حداکثر تأخیر (ms)
- ✅ مرتب‌سازی نتایج بر اساس سرعت (سریع‌ترین اول)
- ✅ رنگ‌بندی بصری بر اساس کیفیت:
  - 🟢 سبز = زیر ۵۰ms (عالی)
  - 🟡 زرد = ۵۰ تا ۱۰۰ms (خوب)
  - 🟠 نارنجی = ۱۰۰ تا ۱۲۰ms (قابل قبول)
- ✅ ساخت کانفیگ‌های جدید با جایگزینی IP
- ✅ دانلود خروجی به صورت فایل txt
- ✅ کاملاً فارسی و بهینه برای موبایل

-----

## 🚀 نحوه استفاده

### مرحله ۱ — باز کردن ابزار

به آدرس زیر برو:

```
https://sadra1986.github.io/IP-CF-TESTER---CONFIG-REGENERATE
```

یا فایل `index.html` رو مستقیم روی مرورگرت باز کن.

-----

### مرحله ۲ — وارد کردن لیست IP

دو روش داری:

**روش الف — بارگذاری خودکار از IRCF:**
دکمه **☁️ بارگذاری لیست IRCF** رو بزن تا لیست آپ‌تودیت IP های تمیز Cloudflare دریافت بشه.

**روش ب — آپلود دستی:**
فایل txt حاوی IP ها رو (هر IP در یک خط) آپلود کن یا paste کن:

```
104.21.72.109
69.84.182.49
172.67.200.192
...
```

-----

### مرحله ۳ — تنظیمات تست

|تنظیم    |توضیح                           |پیش‌فرض |
|---------|--------------------------------|-------|
|تایم‌اوت  |حداکثر زمان انتظار برای هر IP   |۵ ثانیه|
|همزمان   |تعداد IP هایی که همزمان تست میشن|۵      |
|حداکثر ms|IP هایی با تأخیر بالاتر حذف میشن|۱۲۰ms  |

-----

### مرحله ۴ — شروع تست

دکمه **▶ شروع تست** رو بزن. نتایج real-time نمایش داده میشن و به صورت خودکار مرتب میشن.

-----

### مرحله ۵ — ساخت کانفیگ جدید

۱. کانفیگ پایه‌ات رو توی باکس **⚙️ کانفیگ پایه** paste کن:

```
vless://UUID@1.2.3.4:443?type=ws&...#name
```

۲. بعد از اتمام تست، باکس **🔧 ساخت کانفیگ** ظاهر میشه

۳. تعداد کانفیگ مورد نظر رو بنویس (مثلاً ۱۰)

۴. دکمه **⚡ بساز و دانلود** رو بزن

فایل خروجی `generated-configs.txt` شامل کانفیگ‌هایی میشه که:

- IP هر کدوم با سریع‌ترین IP های پیدا شده جایگزین شده
- مرتب بر اساس سرعت (سریع‌ترین اول)
- اسم هر کانفیگ شامل شماره و ms هست (مثلاً `name-1-45ms`)

-----

### مرحله ۶ — ایمپورت در فیلترشکن

فایل خروجی رو میتونی:

- مستقیم توی فیلترشکن ایمپورت کنی
- یا توی GitHub Gist آپلود کنی و لینک Raw رو به عنوان **ساب‌اسکریپشن** بدی

-----

## 📡 ساخت ساب‌لینک (اختیاری)

برای اینکه کانفیگ‌هات رو به دوستات بدی یا فیلترشکنت رو آپدیت نگه داری:

**۱. GitHub Gist (Secret):**

- برو `gist.github.com`
- فایل `configs.txt` بساز و کانفیگ‌ها رو paste کن
- **Create secret gist** بزن
- روی **Raw** کلیک کن — این لینک = ساب‌لینکته

**۲. Cloudflare Worker (برای دور زدن فیلتر):**
اگه لینک Gist فیلتر بود، یه Worker رایگان روی `workers.dev` بساز که درخواست رو به Gist پروکسی کنه.

## 🔧 تکنولوژی

- خالص HTML + CSS + JavaScript
- بدون هیچ dependency خارجی
- تست از طریق HTTP probe (image trick) — چون مرورگر اجازه ICMP ping نمیده
- کاملاً client-side — هیچ داده‌ای به سرور ارسال نمیشه

-----

## ⚠️ نکات مهم

- تست‌ها با **نت خودت** انجام میشه — نتایج بر اساس ISP و موقعیت تو متفاوته
- IP هایی که جواب میدن لزوماً برای همه کار نمیکنن — با نت‌های مختلف تست متفاوته
- این ابزار فقط **دسترسی HTTP** رو تست میکنه، نه اتصال کامل VPN
- برای بهترین نتیجه، تایم‌اوت رو روی ۵ ثانیه نگه دار
